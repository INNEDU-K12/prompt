Học cách viết prompt tốt qua tương tác

8 prompt mẫu trên 6 module của khóa học. Mỗi prompt được trình bày như một bài học tương tác gồm 4 giai đoạn: hiểu cấu trúc → điền form → nhận prompt hoàn chỉnh → đọc phản hồi tự động theo khung FEED.

  1. Chọn một prompt ở thanh bên trái.
  2. Đọc lý thuyết và cấu trúc để hiểu cách prompt hoạt động.
  3. Điền form — prompt sẽ tự cập nhật theo thời gian thực.
  4. Đọc phản hồi FEED và tinh chỉnh để prompt tốt hơn.
